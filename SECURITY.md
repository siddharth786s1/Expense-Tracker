# Security Policy

## Supported Versions

Only the latest release is supported with security fixes.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Instead, report privately by emailing: security@example.com

Include:
- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

We will acknowledge receipt within 72 hours and provide updates during investigation.
